# config.go

Manages loading and saving of akb's YAML configuration file (`docs/akb/.config.yaml`), which stores source file extensions and exclude patterns for repository analysis.

## Types

### Config

```go
type Config struct {
    SourceExtensions []string `yaml:"source_extensions"`
    ExcludePatterns  []string `yaml:"exclude_patterns"`
}
```

Configuration data for akb. `SourceExtensions` lists file extensions to analyze (e.g., `.go`, `.ts`). `ExcludePatterns` lists glob patterns for files/directories to skip.

## Functions

### Path

```go
func Path(repoPath string) string
```

Returns the absolute path to the config file: `<repoPath>/docs/akb/.config.yaml`.

- **Parameters:** `repoPath` — root of the target repository
- **Returns:** config file path as a string

### Exists

```go
func Exists(repoPath string) bool
```

Checks whether the config file exists on disk. Uses `os.Stat` on the path returned by `Path()`.

- **Parameters:** `repoPath` — root of the target repository
- **Returns:** `true` if the config file exists, `false` otherwise

### Load

```go
func Load(repoPath string) (*Config, error)
```

Reads and parses the YAML config file. Validates that `SourceExtensions` is non-empty, returning an error with remediation guidance if missing.

- **Parameters:** `repoPath` — root of the target repository
- **Returns:** parsed `*Config` or an error (file read failure, YAML parse failure, or empty `source_extensions`)
- **Dependencies:** `gopkg.in/yaml.v3`

### Save

```go
func Save(repoPath string, cfg *Config) error
```

Marshals the config to YAML and writes it to disk. Creates the parent directory (`docs/akb/`) if it doesn't exist. Prepends a comment header (`# Generated by akb init. Edit as needed.`) to the output file.

- **Parameters:** `repoPath` — root of the target repository; `cfg` — configuration to persist
- **Returns:** error on directory creation, marshal, or write failure
- **Side effects:** creates directories, writes file at `docs/akb/.config.yaml`
- **Note:** does not use atomic writes (direct `os.WriteFile`), unlike some other write operations in the codebase
- **Dependencies:** `gopkg.in/yaml.v3`
